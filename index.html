Rygel turned my Raspi3 into a DLNA movie player

Mar 24, 2019

Installing Rygel and VLC on a raspbian is enough to remotely control with a smartphone the streaming of movies from a media server to a Raspberry Pi 3. No more need to install full-fledged, heavyweight media centers.
